# CodeAlpha Frontend Development Tasks

This repository contains the frontend development tasks completed as part of the CodeAlpha Internship.

## Projects

### 1. Calculator
- Basic calculator using HTML, CSS, and JavaScript.
- Performs arithmetic operations with a responsive interface.

### 2. Image Gallery
- Responsive image gallery.
- Clean layout with interactive image display.

### 3. Music Player
- Simple music player built with HTML, CSS, and JavaScript.
- Includes play, pause, next, and previous controls.
